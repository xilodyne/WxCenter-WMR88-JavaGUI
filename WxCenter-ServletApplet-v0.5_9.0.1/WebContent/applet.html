Casa Holiday Weather Center Applet

This application is displaying an Oregon Scientific WMR88 Weather Center.
It reads the WMR88 console using WxLogger WxLogger by Ken Turner.
It uses some of Gerrit Grunwald's Steel Series dials.


To allow the WxCenterServer applet to run, add the following lines to the C:\Program Files (x86)\Java\jre7\lib\security\java.policy

  • add the following line:
  • permission java.lang.reflect.ReflectPermission "suppressAccessChecks";
  • permission java.util.PropertyPermission "mapAnyUriToUri", "read";
  • permission java.lang.RuntimePermission "accessDeclaredMembers";
  • permission java.lang.RuntimePermission "exitVM.0";
  • set the Java control panel, security tab to Medium (allow self-signed certificates)
